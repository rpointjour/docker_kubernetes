## Structure

### Tutorial/
Introduction to Docker using Docker's official `welcome-to-docker` image. Contains a `docker-compose.yml` that spins up a local nginx web server on port `8088`.

```bash
cd Tutorial
docker compose up
```

Then open `http://localhost:8088` in your browser.
